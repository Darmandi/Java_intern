Что такое lamdba выражение?

Лямбда-выражение представляет собой блок кода, который можно передать в другое место, поэтому он может быть выполнен позже, один или несколько раз.

Что такое функциональные интерфейсы?

Функциональный интерфейс в Java – это интерфейс, который содержит только 1 абстрактный метод. Основное назначение – использование в лямбда выражениях и method reference.

Перечислите функциональные интерфейсы из пакета java.util.function.

Package java.util.function

Их можно разделить на 4 группы:

  • Consumer - принимает параметры, метод ничего не возвращает void.
  • Function - принимает параметры и возвращает значение.
  • Predicate - принимает параметры и возвращает значение типа boolean. Используется для фильтрации данных.
  • Supplier - не принимает параметры, возвращает значение.

Что такое функции высшего порядка?

Функциями высшего порядка— это функции, которые передают функции в качестве параметров или возвращают функции в качестве значений. Результатом является гораздо более гибкий стиль программирования, чем в случае лямбда-выражений, включая возможность использования функций в качестве стандартных блоков для составления других функций.

Что такое ссылки на методы?

Ссылка на метод эквивалентна лямбда-выражению. Оператор :: отделяет имя метода от имени объекта или класса. Есть три основных варианта:

  • object::instanceMethod
  • Class::staticMethod

Расскажите о зоне видимости переменных в lambda - выражениях?

Внутри лямбда-выражений мы имеем доступ к:

  • константам, объявленым внутри метода, содержащего дямбда-выражение. При условии, что константа объявлена да лямда-выражения. Внутри лямда-выражения изменить значение константы нельзя.
  • локальным переменным, объявленным внутри метода, содержащего лямбда-выражение, которые не менияют своего значения после инициализации. При условии, что локальная переменная объявлена до лямбда-выражения. Изменить ее внутри лямбда-выражения нельзя.
  • статическим методам.
  • статическим переменным класса, константам класса. Внутри лямбда-выражения можно изменить статическую переменную класса.
  • полям и методам зкземпляра, если лямбда-выражение создается внутри обычного метода. Внутри лямбда-выражения можно изменить значение поля.

Что такое Stream API?

Stream API это новый способ работать со структурами данных в функциональном стиле. Чаще всего с помощью stream в Java 8 работают с коллекциями.

Что делает метод filter?

Отфильтровывает записи, возвращает только записи, соответствующие условию

collection.stream().filter(«a1»::equals).count()

Что делает метод map?

Преобразует каждый элемент стрима

collection.stream().map((s) -> s + "_1").collect(Collectors.toList())

Что делает метод flatMap?

Похоже на map, но может создавать из одного элемента несколько

collection.stream().flatMap((p) -> Arrays.asList(p.split(",")).stream()).toArray(String[]::new)

Что делает метод collect?

Представление результатов в виде коллекций и других структур данных

collection.stream().filter((s) -> s.contains(«1»)).collect(Collectors.toList())

Что делает метод reduce?

Позволяет выполнять агрегатные функции на всей коллекцией и возвращать один результат

collection.stream().reduce((s1, s2) -> s1 + s2).orElse(0)

Что делает метод findFirst?

Возвращает первый элемент из стрима

collection.stream().findFirst().orElse(«1»)

Возможно ли прервать выполнение потока по аналогии с break?

Stream.takingWhile - это метод позволяет получать поток данных до тех пор, пока он проходит фильтр.

Возможно ли пропустить элемент потока по аналогии с continue?

collection.stream().skip(N) - позволяет пропустить N первых элементов.

Что такое Optional?

Optional - класс-обертка. Optional — ссылочный тип, он может быть null. Может находиться в 2ух состояниях:

  • Содержит ссылку на T (также «present» или присутствует)
  • Пуст (также «absent» или отсутствует. Не употребляйте «null»)

Для чего используется ключевое слово var?

При объявлении локальных переменных вы можете использовать var вместо имени класса или интерфейса, чтобы позволить компилятору вывести тип.
Это работает только в том случае, если переменная немедленно инициализируется, например, как в var s = "".
Индексы для циклов также могут быть объявлены с помощью var.
Тип, выводимый компилятором, помещается в байт-код, поэтому во время выполнения ничего не меняется — Java все еще является статически типизированным языком.
Технически, var — это не ключевое слово, а зарезервированное имя типа, то есть его можно использовать только в тех местах, где компилятор ожидает имя типа, но во всех остальных местах он является допустимым идентификатором.
Это означает, что только классы, называемые var, больше не будут работать, но это не особо частый случай.

В какие случаях можно использовать var?

Помимо локальных переменных, например в полях и сигнатурах методов, var не может применяться.
