Что такое виртуальная машина?

ВМ - система, эмулирующая некую платформу. В Java предназначена для исполнения байт-кода на любом железе, в независимости от его спецификации.

К какому типа языка программирование относиться Java?

Java - объектно-ориентированный язык, что значит, что все элементы программ являются объектами.
Однако элементарные типы данных остаются высокопроизводительными компонентами, не являющимися объектами.

Из каких компонентов состоит Java (JDK, JRE, JVM)?

Для чего используется JDK?

JDK - инструмент для разработки программ.

Для чего используется JRE?

JRE - комплект для исполнения программ, без компилятора, но с ВМ.

Для чего используется VM?

VM - интерпретирует байт-код и исполняет его на конечном устроййстве. Может изолировать программу и воспрепятствовать побочным эффектам за пределами VM.

Что такое ссылочные типы?

Все остальные типы являются ссылочными типами: они ссылаются на объекты.

Что такое классы обертки?

Примитивные типы используются из соображений производительности. Когда необходимо оперировать простыми значениями, как обьектами, используются классы-обертки, которые заключают примитивный тип в оболочку обьекта.
Double, Float, Integer, Long, Short, Byte, Character, Boolean
